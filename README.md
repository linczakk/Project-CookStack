# Project-CookStack
Personal cooking assistant web app for managing recipes and shopping lists.


CookStack is a full-stack web application built as a portfolio project to demonstrate clean architecture, API design, and modern .NET web development.

Features
- Recipe and shopping list CRUD (ingredients, steps, units)
- Shopping list generated from recipes (planed)
- Progress tracking while cooking (planed)
- Blazor WASM client with ASP.NET Core API
- Unit tests (planned)

Tech Stack
- .NET 10
- ASP.NET Core Web API
- Blazor WebAssembly
- Entity Framework Core
- Temporary SQL Server Express LocalDB. In futer SQLLite/PostgreSQL
- Bootstrap

Architecture 
- Clean-ish layered architecture
- DTOs and mapping
- Client-server separation
- Centralized routes and services
